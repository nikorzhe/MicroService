<b>Для початку потрібно створити .env файл та задати значення цим змінним :
SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_USERNAME=email@gmail.com
SMTP_PASSWORD=password

<b>Запуск
Запуск виконується командою - docker compose up -d --build
Варто зауважити, що для бекенду Postgresql потрібно запускати локально
